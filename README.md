## Reproducing results

### Prerequisites

- Python 3.7+ (with Pip)

### Setup

We advise you to use a [virtual environment](https://docs.python.org/3/library/venv.html) for running the code. After
you activated it change to the `source` directory and run:

```sh
pip install -r requirements.txt
```
